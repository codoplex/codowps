== Description ==
CODO WP Plugins helps list and filter all WordPress plugins you've developed, hosted on Codecanyon, WordPress.org, or GitHub.

== Installation ==
1. Upload `codowps` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[codowps_plugin_showcase]` shortcode on any page.
